# Required Engineering Skills

To maintain and extend this MVP into a fully productionized system, the team requires the following matrix:

## Core Technical Skills
* **C++ / C#:** For high-performance edge computing and robust microservice APIs.
* **Kubernetes (K8s) & K3s:** Managing multi-cluster environments and edge deployments.
* **Infrastructure as Code (IaC):** Terraform for provisioning cloud and on-prem hardware identically.
* **Time-Series Databases:** Familiarity with TimescaleDB or InfluxDB for storing IoT payloads.
* **Cybersecurity Principles:** Managing PKI infrastructure, mTLS, and network isolation in industrial environments.
